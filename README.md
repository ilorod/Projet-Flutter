# ClickNum

ClickNum est un mini-jeu Flutter où l'on doit appuyer sur les chiffres dans l'ordre le plus rapidement possible.

## Fonctionnalités
- Chrono par niveau
- Niveaux progressifs
- Boutons dispersés
- Verrouillage de niveaux
- Page de sélection des niveaux

## Lancement
```bash
flutter pub get
flutter run
